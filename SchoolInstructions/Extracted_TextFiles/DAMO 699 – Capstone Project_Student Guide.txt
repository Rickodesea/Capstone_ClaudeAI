DAMO 699 – Capstone Project 
Student Guide  
Student Guide and Project Timeline 
The capstone project is a term-long group research and analytics project. The project progresses 
through several stages, including proposal development, preliminary analysis, draft reporting, and 
final presentation. As a group, you are expected to maintain an active GitHub project repository 
throughout the course. Supervisors will review the repository to monitor progress and provide 
feedback. 
Expected Learning Outcomes 
By completing the capstone project, students will demonstrate the ability to: 
1. define a research or analytical problem 
2. collect and prepare data for analysis 
3. apply appropriate analytical or machine learning methods 
4. interpret results and communicate insights 
5. collaborate effectively within a team 
6. present findings through written reports and professional presentations

GitHub Repository Requirement 
Each group must maintain a GitHub project repository throughout the course. 

The repository should include: 
• 
datasets (when permitted) 
• 
cleaned datasets 
• 
code used for analysis 
• 
notebooks or scripts 
• 
data visualizations 
• 
project documentation 
• 
draft and final reports 
Supervisors will periodically review the repository to evaluate progress and provide feedback. 
Maintaining a well-organized repository is part of professional data analytics practice and 
ensures that the project is reproducible and transparent.
